## 第一步：登录系统

1. 打开浏览器，访问系统地址
2. 输入您的账号密码登录

---

## 第二步：生成第一份文档

### 最简单的方式

直接在聊天框输入：

```
请帮我生成技术交底书：基于物联网的智能绿植监测系统
```

### 稍详细的方式（推荐）

```
请生成技术交底书：

发明名称：基于物联网的智能绿植监测系统

技术领域：农业物联网

技术背景：
现有绿植养护依赖人工经验，无法及时响应植物需求...

要解决的技术问题：
如何实时监测植物状态并自动提供养护建议

技术方案：
采用土壤传感器、光照传感器、温湿度传感器采集数据，
通过边缘计算分析植物健康状态，自动控制灌溉设备...

有益效果：
1. 实现精准灌溉，节约用水
2. 及时发现病虫害
3. 降低养护人力成本
```

---

## 第三步：查看结果

1. 等待 AI 生成完成
2. 点击右侧 **fiels** 标签
3. 展开 `generated_docs/` 目录
4. 找到生成的文件点击查看

## 常用操作速查

### 我想生成其他文档

| 需求 | 输入示例 |
|------|----------|
| 权利要求书 | "请生成权利要求书：[你的技术方案]" |
| 商业价值分析 | "请做专利商业价值分析：[你的创意]" |
| IP保护策略 | "请生成IP保护策略：[你的项目]" |
| 技术方案 | "请生成技术实现方案：[你的需求]" |

### 我想引用项目文件

```
请分析 @docs/需求文档.md 中的需求，生成技术方案
```

### 我想上传文件处理

1. 点击输入框旁的 📎 图标
2. 选择文件上传
3. 然后告诉 AI 你的需求
